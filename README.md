# Bloom - Floral & Gifts E-Commerce Platform

A customized, production-ready boutique e-commerce platform crafted for a local handmade floral brand. This project completely reimagines the traditional e-commerce workflow by introducing a friction-free checkout and a completely bespoke management system.

## Key Features

* **WhatsApp Checkout Protocol:** Replaced standard, high-friction cart funnels with a dynamic direct checkout that compiles order items, pricing, and client details into a clean WhatsApp message for immediate vendor fulfillment.
* **Bespoke Admin Dashboard:** Designed and developed a tailored management interface from scratch, completely decoupling day-to-day operations from Django's default admin panel.
* **Touch-Responsive UI:** Built an intuitive, visual-heavy image gallery optimized for showcasing premium products fluidly across all devices.

## Tech Stack

* **Backend:** Python, Django Web Framework, Django ORM
* **Database:** PostgreSQL (Hosted via Neon) / SQLite (Local)
* **Frontend:** Tailwind CSS, Native JavaScript, HTML5 Semantic Tags
* **Deployment:** Vercel (Serverless Architecture)
